Image Guidelines
Including Images

To include images in questions or answers, use the simple IMAGE: syntax:

IMAGE: filename.png
Image Requirements
  • Upload images using the "Upload Images" button above
  • Supported formats: PNG, JPG, JPEG, GIF
  • Maximum file size: 5MB
  • Recommended: max 800px width for best display
