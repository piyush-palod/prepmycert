How to Use Images in Questions
Method 1: Simple Syntax (Recommended)

Use this format in your question text:

What is shown in this diagram? IMAGE: your-filename.png

The system will automatically convert this to Azure URLs with SAS tokens when you save the question.

Method 2: Direct HTML (Advanced)

Use standard HTML img tags (system will process these too):

<img src="any-path/filename.png" alt="Description" class="img-fluid">

The system will automatically update any img tags with the correct Azure URL and SAS token.

Important Notes:
  • Processing: Images are processed once when you create/edit questions - no processing during test-taking!
  • Security: Images are secured with 30-day SAS tokens and only accessible to students who purchased the course
  • Performance: Images load instantly during tests since URLs are pre-processed
  • File Size: Keep images under 2MB for optimal loading speed
