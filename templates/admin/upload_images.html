How to Use Uploaded Images

After uploading an image, you can reference it in your questions or answer options using standard HTML img tags:

Basic Usage:
<img src="/static/images/questions/{{ package.title|lower|replace(' ', '_')|replace('[^a-zA-Z0-9\-_]', '_', true) }}/your-image.png" alt="Description">
Recommended HTML with Styling:
<img src="/static/images/questions/{{ package.title|lower|replace(' ', '_')|replace('[^a-zA-Z0-9\-_]', '_', true) }}/your-image.png" 
     alt="Description" 
     class="img-fluid" 
     style="max-width: 100%; height: auto; margin: 10px 0;">
Tips for Better Images:
  • Use descriptive filenames (e.g., aws-architecture-diagram.png)
  • Always include meaningful alt attributes for accessibility
  • Keep image sizes reasonable (< 1MB when possible)
  • Use class="img-fluid" for responsive images
